# Anticipate vs. Respond Modes

![Anticipate vs Respond Flowchart](file:///c:/Users/admin/agy2-projects/golden-hour/docs/anticipate-vs-respond.png)

## Anticipate Mode
Used for: Floods and Cyclones
Why: These disasters can be forecast days ahead using weather models and river gauge data. Golden Hour acts BEFORE the disaster hits.
Data: NOAA NWPS river forecasts, GDACS flood and cyclone alerts

## Respond Mode  
Used for: Earthquakes and Tsunamis
Why: These cannot be predicted. No science exists to forecast when or where an earthquake will strike. Golden Hour acts the MOMENT one is detected using USGS PAGER impact data.
Data: USGS GeoJSON, USGS PAGER, GDACS earthquake alerts

## The Key Difference
- **Anticipate Mode** = Days of warning → Pre-position resources
- **Respond Mode** = Minutes after detection → Rapid situation report
